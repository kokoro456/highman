## 2. 디자인 요청 범위

### 필요한 디자인 산출물

- [ ] 전체 디자인 시스템 (색상, 타이포그래피, 컴포넌트)
- [ ] 22개 페이지 UI/UX 디자인 (Figma)
- [ ] 모바일 반응형 디자인
- [ ] 다크모드 디자인
- [ ] 아이콘 & 로고 디자인
- [ ] B2C 고객 예약 페이지 디자인 (모바일 퍼스트)

### 페이지 목록

#### B2B (매장 관리자용) — 16개 페이지
1. **로그인** — 이메일/소셜 로그인 (카카오, 네이버)
2. **회원가입** — 이름, 이메일, 전화번호, 비밀번호
3. **대시보드** — 오늘 매출/예약/고객 요약, 외부 연동 상태
4. **예약 관리** — 캘린더 뷰 (데스크탑) + 리스트 뷰 (모바일), 예약 등록 모달
5. **고객 관리** — 고객 목록 (검색/필터/태그), 고객 등록 모달
6. **고객 상세** — 시술 내역 타임라인, 시술 사진 갤러리, VIP 등급 표시
7. **매출/결제** — 일별 매출 요약, 결제 내역, 결제 등록 모달
8. **보고서** — 매출/서비스/고객/시간대 분석 (4탭), CSS 차트
9. **쿠폰 관리** — 쿠폰 목록, 쿠폰 생성 모달
10. **직원 관리** — 직원 카드, 스케줄, 전문 분야
11. **정산 관리** — 월별 직원 급여/인센티브 정산
12. **재고 관리** — 재고 목록, 입출고 기록, 부족 알림
13. **매장 설정** — 매장 정보, 영업시간, 서비스 메뉴, 외부 연동
14. **관리자 (슈퍼어드민)** — 전체 매장 통계, 매장 목록
15. **온라인 결제 (체크아웃)** — 토스페이먼츠 결제 화면
16. **결제 성공/실패** — 결제 결과 표시

#### B2C (고객용) — 1개 페이지
17. **온라인 예약** — 5단계 예약 플로우 (서비스→직원→날짜/시간→정보입력→확인)

- **프리미엄/하이엔드** — 뷰티 산업에 어울리는 세련된 디자인
- **직관적** — IT에 익숙하지 않은 원장님도 바로 사용 가능
- **깔끔한 데이터 표현** — 매출/예약/고객 데이터를 한눈에
- **모바일 최적화** — 매장에서 태블릿/폰으로 주로 사용

### 벤치마킹 레퍼런스
1. **공비서** — https://home.ahasoft.co.kr (뷰티 매장 관리 앱)
2. **네이버 예약 관리** — 네이버 스마트플레이스 예약 관리 화면
3. **토스 비즈니스** — 깔끔한 매출 대시보드
4. **Linear** — 미니멀 SaaS 디자인

## 6. 주요 사용 시나리오

### 시나리오 1: 원장님의 하루
1. 아침에 대시보드에서 오늘 예약 확인
2. 고객 방문 → 예약 상태 변경 (확정→진행중→완료)
3. 시술 후 결제 처리 (카드/현금/회원권)
4. 퇴근 전 일일 매출 확인

### 시나리오 2: 고객의 예약
1. 카카오톡/네이버로 매장 B2C 예약 페이지 접속
2. 원하는 서비스 선택 → 날짜/시간 선택
3. 이름/전화번호 입력 → 예약 완료
4. 카카오 알림톡으로 예약 확인 수신

### 시나리오 3: 월말 정산
1. 정산 관리 페이지에서 월별 직원 성과 확인
2. 인센티브 자동 계산 결과 검토
3. 엑셀 내보내기로 급여 데이터 다운로드

## 7. 특별 요청사항

1. **예약 캘린더**가 가장 중요한 화면 — 직원별 컬럼, 시간대별 블록이 한눈에 보여야 함
2. **고객 VIP 등급** 시각적으로 잘 구분되어야 함 (일반/실버/골드/VIP/VVIP)
3. **다크모드** 기본 지원 — 밤에 매장에서 사용 시 눈 피로 감소
4. **B2C 예약 페이지**는 고객이 사용하므로 특히 세련되어야 함
5. **로딩/빈 상태/에러 상태** 각각의 디자인도 포함해주세요
6. **토스트 알림** 디자인 (성공/에러/정보)
